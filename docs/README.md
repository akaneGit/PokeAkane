# PokeAkane ドキュメント

## 📚 ドキュメント一覧

### 📋 **[FEATURES_SPEC.md](FEATURES_SPEC.md)**
- **内容**: 全機能の詳細仕様書
- **対象**: 開発者・機能理解
- **概要**: 1025体のポケモンデータ、34図鑑対応、検索・フィルター機能の完全仕様

### 🖼️ **[FORMS_ANALYSIS.md](FORMS_ANALYSIS.md)**
- **内容**: フォルム違いポケモンの分析レポート
- **対象**: フォルム管理・画像収集

### 🚀 **[SERVER_SETUP.md](SERVER_SETUP.md)**
- **内容**: サーバー起動・管理ガイド
- **対象**: ユーザー・開発者
- **概要**: バッチファイル起動、コマンドライン操作、サーバー管理の簡単ガイド

### 🔧 **[DEVELOPMENT.md](DEVELOPMENT.md)**

- **対象**: 開発者・コントリビューター
