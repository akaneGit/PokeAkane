# PokeAkane - ポケモン総合図鑑ツール

## 🎀 プロジェクト概要
全1025体のポケモンと多様なフォルムに対応した完全自己完結型のWeb図鑑ツール

## ✨ 主要機能

### 🔍 **検索・フィルター機能**
- ポケモン名検索（日本語・英語名対応）
- 図鑑番号検索（全国・地方図鑑対応）
- 18タイプ別フィルター（複数選択可能）
- 進化段階フィルター
- リアルタイム絞り込み機能

### 📊 **完全なポケモンデータベース**
- **全1025体**のポケモン基本情報
- **6つのフォルム系統**対応（45+種類のフォルム）
- **34種類の図鑑**対応（全国・地方・DLC図鑑）
- タイプ・種族値・進化系統データ

### 🖼️ **フォルム表示システム**
- **リージョナルフォルム**: アローラ（18種）、ガラル（19種）、ヒスイ（17種）、パルデア（5種）
- **特殊フォルム**: メガシンカ、ダイマックス、フォルムチェンジ
- **性別差**: オス・メス差のあるポケモン対応
- **色違い**: 1025体の色違い画像対応
- **パターン**: ビビヨン18種、トリミアン9種類など

### 🎨 **モダンなUI/UX**
- レスポンシブデザイン
- 世代別・図鑑別ナビゲーション
- 高度なソート機能（種族値順・ステータス順）
- フォルム切り替えUI
- タイプ相性表・性格表連携

## 🚀 クイックスタート

### 簡単起動（Windows）
```bash
# バッチファイルで起動
server.bat

# または直接起動
python tools/server_manager.py start
```

```

```
