## 🚀 **使用方法**

### **🖥️ ローカル起動**
```bash
# プロジェクトディレクトリに移動
cd PokeAkane

# HTTPサーバー起動
python -m http.server 8000

# ブラウザでアクセス
http://localhost:8000/pokemon_gallery.html
```

### **🔍 基本操作**
1. **世代選択**: 上部の世代ボタンで第1〜9世代を選択
2. **図鑑選択**: 図鑑ボタンで全国図鑑・地方図鑑を切り替え
3. **検索**: 検索ボックスでポケモン名・図鑑番号を入力
4. **フィルター**: タイプ・進化段階で絞り込み
5. **ソート**: 種族値順・ステータス順で並び替え
6. **詳細表示**: ポケモンカードをクリックで詳細表示
